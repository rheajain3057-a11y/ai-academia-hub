# Assignment Helper

Assignment Helper is a modern academic workspace designed to help students plan assignments, organize study routines, and stay on top of deadlines.

## Project Structure

The active app source is located in the main project root:

- [src/App.jsx](src/App.jsx) — main landing page and UI structure
- [src/App.css](src/App.css) — landing page styling
- [src/index.css](src/index.css) — global base styles
